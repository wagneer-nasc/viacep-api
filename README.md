 ## 📁 Como baixar o projeto

```bash
# clonar repositório
$ git clone https://github.com/wagneer-nasc/viacep-api.git

# Entrar no diretório
$ cd viacepapi

# Instalar as dependencias
$ yarn install
$ cd ios pod install

# inicar o projeto
$ yarn ios para quem está com o emulador IOS
$ yarn run android para quem está com emulador Android

```
